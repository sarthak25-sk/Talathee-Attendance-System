# Talathee Attendance System

## Project Description
The Talathee Attendance System is an Android application designed for managing attendance in educational institutions. Built using Kotlin, this app aims to streamline the attendance tracking process, making it easier for teachers and students alike.

## Features
- User authentication for both students and teachers.
- Ability to mark attendance.
- View attendance history for students.
- Generate attendance reports.
- User-friendly interface with easy navigation.
- Notifications for attendance updates.

## Tech Stack
- **Programming Language:** Kotlin
- **Framework:** Android SDK
- **Database:** Firebase Realtime Database
- **Authentication:** Firebase Authentication
- **UI Library:** Jetpack Compose
- **Version Control:** Git

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sarthak25-sk/Talathee-Attendance-System.git
   ```
2. Open the project in Android Studio.
3. Ensure that you have the latest version of the Android SDK installed.
4. Sync the project with Gradle files.
5. Run the application on an emulator or physical device.

## Usage Instructions
- Launch the app after installation.
- Register or log in to access the features.
- Teachers can mark attendance and manage reports.
- Students can view their attendance history directly in the app.
- Use the settings to customize notifications and profiles.
